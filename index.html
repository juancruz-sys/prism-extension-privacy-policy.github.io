Prism Extension Privacy Policy

Last updated: February 23, 2026

Prism is a Chrome extension that helps users copy text entered in internal QA task forms from a supported active page and autofill it into the Prism QA validation form. The extension runs only when the user clicks its actions (Copy / Autofill).

1. Information We Access

Prism may access website content on supported pages (for example, internal form field values and task-related text) only after an explicit user action, such as clicking “Copy QA Data” or “Autofill QA Form”.

  • No background collection or monitoring.
  • No crawling of unrelated websites.
  • No access to other tabs unless the user explicitly uses the extension on that tab.

2. How We Use Information

The accessed text is used only to copy values from the currently active supported QA task page and autofill the Prism QA form. Prism does not use this content for analytics, advertising, profiling, or any unrelated purpose.

3. Local Storage

Prism stores the most recent copied text payload in chrome.storage.local (local storage on the user’s device) to allow users to copy first and autofill later.

This stored text is not transmitted to external servers and is overwritten on each new copy action, or removed if the user clears browser/extension storage.

4. Sharing and Transfers

Prism does not sell, share, or transfer user data to third parties.

Prism does not send copied form text to any external servers.

5. Permissions

Prism requests permissions only to perform its user-initiated Copy/Autofill functionality on supported pages:

  • activeTab: temporary access to the active tab after a user click.
  • tabs: identify the active tab and validate the supported page context.
  • scripting: run scripts in the active tab to read form text and autofill the Prism form.
  • storage: store the most recent copied text locally for later autofill.

6. Intended Audience

This extension is intended for authorized users of the expert.micro1.ai workspace and supported internal QA pages.
