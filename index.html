Prism Extension Privacy Policy

Last updated: February 23, 2026

Prism is a Chrome extension used to copy QA task data from the currently active task page and autofill the Prism QA validation form.

1. Data We Access

Prism may access and read website content (such as form field values and task-related text on supported pages) only when the user explicitly clicks the extension action (for example, “Copy QA Data” or “Autofill QA Form”).

Prism does not collect data automatically in the background and does not crawl unrelated tabs or websites.

2. How We Use Data

The data read from supported pages is used solely to copy QA task values from the active page and autofill the Prism QA validation form. This use is limited to the extension’s single purpose.

3. Storage

Prism stores the most recent copied QA payload in chrome.storage.local (local browser storage on the user’s device) so the user can copy data first and autofill later.

Data remains on the device unless the user clears browser storage or overwrites the stored payload by using the extension again.

4. Data Sharing and Transfer

Prism does not sell, share, or transfer user data to third parties.

Prism does not transmit copied QA data to external servers.

5. Permissions

Prism uses Chrome extension permissions (activeTab, tabs, scripting, and storage) only as necessary to perform its single purpose after user interaction.

6. Availability

This extension is intended for authorized members of the expert.micro1.ai workspace.
